C 언어

C 언어는 범용 프로그래밍 언어로, 시스템 프로그래밍 및 임베디드 시스템에서 널리 사용됩니다. C는 효율성과 저수준 메모리 접근을 강조하는 언어입니다.

C#

C#는 Microsoft에서 개발한 객체 지향 프로그래밍 언어로, .NET 프레임워크에서 주로 사용됩니다. C#는 강력한 타입 시스템과 풍부한 라이브러리를 제공하여 윈도우 응용 프로그램, 웹 애플리케이션, 게임 개발 등에 사용됩니다.

C++

C++는 C 언어를 기반으로 한 객체 지향 프로그래밍 언어로, 성능과 효율성을 강조하면서도 높은 수준의 추상화를 제공합니다. C++는 게임 개발, 시스템 소프트웨어, 드라이버 등 다양한 분야에서 사용됩니다.

Java
